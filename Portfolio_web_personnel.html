Portfolio Web Personnel

HTML CSS Design moderne One Page

Contexte & objectif

Dans le cadre de ma formation en BTS SIO SISR, j’ai conçu et développé un portfolio web personnel afin de présenter mon parcours, mes compétences et mes projets.

L’objectif principal était de créer un site moderne, fluide et professionnel permettant de valoriser mon profil auprès de recruteurs ou d’entreprises.

Réalisation

Structure HTML

Création d’une structure claire en une seule page avec différentes sections : accueil, à propos, compétences et projets.

Design CSS moderne

Mise en place d’un design sombre avec des couleurs modernes, animations et effets visuels pour améliorer l’expérience utilisateur.

Navigation fluide

Utilisation du scroll fluide pour naviguer entre les sections sans rechargement de page.

Responsive design

Adaptation du site pour mobile, tablette et ordinateur.

Compétences mobilisées

Développement web

Maîtrise du HTML et CSS pour la création d’interfaces web modernes.

Design UI/UX

Création d’une interface claire, lisible et agréable à utiliser.

Valorisation professionnelle

Mise en avant des projets et compétences dans un objectif d’insertion professionnelle.

Résultat

Ce portfolio constitue une vitrine professionnelle me permettant de présenter efficacement mon parcours en BTS SIO SISR, mes compétences techniques et mes réalisations.
